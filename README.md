# Python Expert Quiz App

A professional desktop quiz application built with Python.

## Features
- **Modern UI:** Built with CustomTkinter.
- **Timed Challenges:** 15 seconds per question.
- **Health System:** 3 lives to complete the quiz.
- **Results:** Final score summary at the end.

## How to Run
1. Install requirements: `pip install customtkinter.`
2. Run the app: `python your_filename.py.`
